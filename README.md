# Hamza Web Solutions — Portfolio Website

A premium, static HTML/CSS/JS portfolio website for **Muhammad Hamza Hasan** — Founder & Lead Web Developer at Hamza Web Solutions.

## How to Open

1. Extract the ZIP archive.
2. Open `index.html` in any modern browser.
3. No server, no build step, no dependencies to install.

> **Tip:** For the smoothest experience, open via a local server (e.g. VS Code Live Server). Some browsers restrict certain features when opening `file://` URLs directly.

## Pages

| File | Page |
|---|---|
| `index.html` | Home — hero, stats, services preview, CTA |
| `about.html` | About — founder photo, bio, skills |
| `services.html` | Services — 6 service cards |
| `portfolio.html` | Portfolio — filterable project grid + detail modals |
| `process.html` | Process — 6-step timeline with scroll-driven highlighting |
| `testimonials.html` | Testimonials — auto-advancing carousel |
| `contact.html` | Contact — info + contact form |

## Tech Stack

- **HTML5** — semantic, accessible markup
- **CSS3** — custom properties, glassmorphism, CSS animations
- **Vanilla JavaScript ES6+** — no frameworks
- **Font Awesome 6** — icons (CDN)
- **Google Fonts** — Inter typeface (CDN)
- **Lenis** — premium smooth scrolling (CDN, v1.0.42)

## Features

- ✅ Lenis premium smooth scrolling (inertia-based, natural feel)
- ✅ Light / Dark mode toggle (saved to localStorage)
- ✅ Loading screen with animated progress bar
- ✅ Sticky glassmorphism navbar
- ✅ Responsive design (mobile, tablet, desktop)
- ✅ Scroll-reveal animations (IntersectionObserver)
- ✅ Animated stat counters
- ✅ Portfolio filter + project detail modals
- ✅ 6-step process timeline with scroll-driven active state
- ✅ Auto-advancing testimonials carousel
- ✅ Contact form with success state
- ✅ `prefers-reduced-motion` accessibility support
